# MiniPascal
Implémentation d’un compilateur minimal pour un sous-ensemble du langage Pascal, incluant l’analyse lexicale, syntaxique et sémantique.  

# Intructions
- Cloner le dépôt [https://github.com/byt3Buster/MiniPascal_INF325.git](https://github.com/byt3Buster/MiniPascal_INF325.git)  
- Exécuter la commande `chmod u+x mpascal.py`  
- Le fichier *program.mpas* est un exemple de program simple en pascal. Pour l'exécuter, taper `./mpascal program.py --run`  

## Options:
- -h, --help  show this help message and exit  
- --tokens    génère un fichier .tok  
- --ast       génère un fichier .ast  
- --run       exécute l'interpréteur
